what is internet?

the internet is a global network of interconnected computers and servers that communicate using standardized protocols

browsing websites and sending mails

what is browser?

A browser is a software application used to access and view websites on the Internet.

Example:google chrome,firefox

what is URL?

A URL (Uniform Resource Locator) is the address used to access a specific resource on the Internet.

EXample: https://www.wikipedia.org

what is http?

HTTP stands for HyperText Transfer Protocol. It is the set of rules that web browsers and web servers use to communicate and transfer data over the Internet

example:http://www.example.com

what is https?

HTTPS stands for HyperText Transfer Protocol Secure. It is a secure version of HTTP. It encrypts the data sent between your browser and the website to protect it from hackers and unauthorized access.

example:https://www.google.com

what is webapp and website?

A web app is a website with interactive features. It lets users perform tasks online, such as filling forms, making purchases.

example:www.amazon.com

A website is a collection of web pages that you can visit using a browser. It usually contains information, images, videos, and links.

example:www.wikipedia.org

what is domain and divisions of domain?

A domain is the unique name used to identify a website on the Internet.

there are 3 types

sub domain

main domain

top level domain

example:www.google.com

www is a sub domain

google is a main domain

.com is top level domain
